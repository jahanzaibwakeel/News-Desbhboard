# News Dashboard

News Dashboard is a web application that fetches and displays the latest news articles from an external news API.

Users can browse different categories of news in a dashboard-style layout.

## Features

* Fetch latest news articles
* Category filters
* Article cards
* Responsive layout

## Tech Stack

* React
* JavaScript
* News API
* CSS

## Future Improvements

* Search news articles
* Bookmark articles
* Dark mode
* Pagination
